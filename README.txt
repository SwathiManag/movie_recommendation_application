Movie Recommendation Application: This application provides top ten movie recommendations based on the searched movie. It also provides sentiment analysis of each movie review.

How to run the project:
1)download this repository to your local machine.
2)Make sure you have the requirements.txt file in your folder
3)Install the libraries with the command pip install -r requirements.txt
4)Open your terminal/command prompt from your project directory and execute the command python main.py
5)Go to your browser and type http://127.0.0.1:5000/ in the address bar
